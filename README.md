# 🤖 AI Customer Support Chatbot

An intelligent customer support chatbot built for
Jaidatech Cafe Business Centre, powered by Groq API
(LLaMA 3.3-70b) and built with Python, Flask, and
a custom HTML/CSS/JavaScript frontend.

## ✨ Features

- Real-time AI-powered responses
- Custom trained on Jaidatech business information
- Services, location, and hours awareness
- Clean responsive chat interface
- CORS enabled for cross-origin requests

## 🛠️ Tech Stack

- **Python** — Backend logic
- **Flask** — Web framework
- **Groq API** — LLaMA 3.3-70b-versatile model
- **HTML/CSS/JavaScript** — Frontend chat interface
- **Render** — Cloud deployment

## 🚀 How to Run Locally

1. Clone the repository
2. Install dependencies:
pip install -r requirements.txt
3. Create a `.env` file and add:
GROQ_API_KEY=your_groq_api_key_here
4. Run the app:
python app.py
5. Open browser at `http://localhost:5000`
